# LockGroupCard

用于锁定/修改群友的群名片

## 命令

lgcon 开启群名片锁

lgcoff 关闭群名片锁

lgcunlock@ 解锁群名片

lgclock@ 锁定群名片

lgcset@ 修改群名片
